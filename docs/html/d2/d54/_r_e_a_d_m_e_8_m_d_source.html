14 ### Required Processors
15 The following are the processes to be implemented
16 * normalization
17 
18  * Algorithm: The largest sample value in the data is found, and then the data is scaled so that that value is the maximum possible value. This maximizes the amplitude of the final waveform.
19 
20  * Example for floating-point data
21 
22  * Original data: n = {0,0.2,0.4,0,-0.3}
23 
24  * Largest absolute value: 0.4, so scaling value s = 1/0.4 = 2.5
25 
26  * Scaled result: n\_scaled = {0, 0.5,1,0,-0.75}
27 
28 * echo
29 
30  * Algorithm: Samples are copied, scaled, and added to later locations in the sample buffer to create an echo effect.
31 
32 * gain adjustment
33 
34  * Algorithm: Samples are multiplied by a scaling factor that raises or lowers the overall amplitude of the wave file
35 
36 The system design is to be modular to facilitate dividing up the development work, and more importantly, to aid in testing and expansion of the application's capabilities. The required modules are
37 
38 * File management: manages reading and writing of files in the required formats
39 
40 * User interaction: manages the console user interaction. In the case of the challenge version using Qt, the module should provide a single point of entry to the processors, etc.
41 
42 * Processing: management of sample buffers, application of processors, etc. Each module will consist of one or more classes that are access through one or more interfaces.
